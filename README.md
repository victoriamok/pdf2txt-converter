# PDF-TXT-converter

Small and simple pdf-to-txt converter based on the pdfminer package. 

## Requirements

Python (3.6 or above) \
[PDFMiner](https://pypi.org/project/pdfminer/) 

## How to use:

**1. Install PDFminer**
```
pip3 install pdfminer
```
**2. Run the script**
```
python3 pdf2txt.py 
```
